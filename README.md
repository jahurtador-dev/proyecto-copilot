# proyecto-copilot
Proyecto de prueba para Estructura de Datos 2


Instrucciones rápidas:

- Autor: Joan Astrid Hurtado
- Este repositorio contiene un blog técnico con tres posts sobre Arquitectura de Software.

Despliegue en GitHub Pages:

1. Crea un repositorio en GitHub (por ejemplo `proyecto-copilot`).
2. Empuja la rama `main` al repositorio remoto:

```powershell
git init
git add .
git commit -m "Inicial: blog técnico - Joan Astrid Hurtado"
git branch -M main
git remote add origin https://github.com/<tu-usuario>/proyecto-copilot.git
git push -u origin main
```

3. En GitHub, ve a Settings → Pages y elige desplegar desde la rama `main` (carpeta `/ (root)`). Guarda.
4. Tras unos minutos tu sitio estará disponible en `https://<tu-usuario>.github.io/proyecto-copilot/`.

Si quieres, puedo crear un `CNAME`, añadir meta tags SEO, o mejorar el diseño responsivo.
